BrainMapper Help

BrainMapper cnan be used to coregister brain imaging data to create 3D reconstructions with electrode placement and labeled brain regions.

Using BrainMapper

Drag the DICOM image files into the tabs and sort them such that the top image file corresponds with the most dorsal slice.

Drag a folder onto the Target Folder control in the BrainMapper program to provide the location where the resulting images will be stored.

BrainMapper Outputs

When finished, BrainMapper generates three files that can be used to generate a 3D reconstructed model of the brain.
